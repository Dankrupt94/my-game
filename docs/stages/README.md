# Stage Documentation

This folder tracks the staged path from Godot companion app to Godot-AzerothCore-WotLK client.

Each stage file should be updated while work happens:

- status,
- goals,
- deliverables,
- decisions,
- blockers,
- tests,
- next actions.

Status values:

- `Planned`
- `Active`
- `Blocked`
- `Complete`

The master plan is here:

docs/godot-azerothcore-wotlk-master-plan.md
