# Stage Documentation

This folder tracks the staged path from Godot companion app to a fully functional Godot-AzerothCore-WotLK client.

The companion app and Path A sandbox stages are scaffolding only. They are useful work, but they do not satisfy the project mission by themselves. The target remains a Godot-native WotLK client/port that can replace the original WotLK client for normal AzerothCore play.

Each stage file should be updated while work happens:

- status,
- goals,
- deliverables,
- decisions,
- blockers,
- tests,
- next actions.

Status values:

- `Planned`
- `Active`
- `Blocked`
- `Complete`
