# Frostbound Prototype Controls

Open the project in Godot 4 and run the main scene.

## Movement

- `WASD` or arrow keys: move
- Mouse: rotate the camera
- `Space`: jump
- `Esc`: release or recapture the mouse

## MMO Prototype Actions

- `Tab`: target the training dummy
- `1`: Training Strike
- `2`: Frost Spark
- `3`: Mend
- `E`: talk to Scout Mira when close

## First Quest Loop

1. Walk to Scout Mira.
2. Press `E` to accept the quest.
3. Press `Tab` to target the training dummy.
4. Use `1` and `2` to defeat it.
5. Return to Scout Mira and press `E`.
